When someone start making it's own project loses lots of time at making everything the best he/she can, but sometime is better to get some help.

Whith this idea in mind, we wanted to reuse one of the most known icon libraries over the network. Yes, we are talking about FontAwesome.

To use any icon from the font awesome library you just have to include the icon-* class you want. There is a full icon list at the official page.

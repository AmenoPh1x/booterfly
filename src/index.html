Booterfly

Booterfly is a CSS framework that allows you to start your web projects and make them look as awesome as they are.

To do so, we provide tools that will simplify your work. That way you will be able to concentrate on what is more important: Make your dreams come true.

We use Font Awesome that provides with a great amount of icons that will make you application look nice and more fiendly.

Install

In order to use Booterfly in your projects you must include the css and javascript files into them. To do so we provide you many options depending on your preferences.

You can install Booterfly downloading the CSS and JavaScript files and including them into your html or template files.

Install Node and NPM

We will install NodeJS that comes with a package manager named NPM that will help us to install some dependencies on our projects and which is cross-platform.

In order to install NodeJS and NPM you just have to download it from the official page and follow the instructions.

Install SASS

SASS is a CSS precompiler, which means that it add more sintax and features that will help you to increase the basic CSS to save you work and time. You can install it using the following command:

$ gem install sass

Install components

With booterfly we have used some third party components that help us on the developing tasks. To install them into your environment You just have to follow this command into the project folder:

$ npm install

With this command NPM will read package.json file and will install the listed dependences. We list the most importants below:

Bower

Bower is an asset manager. It allows to take control over the assets we need on our projects so we can install or update them easily. We use it to download jquery, font awesome and google code prettify.

Gulp

Gulp is a task manager that helps automatic them so we can concentrate on what is more important.

Developing

If you want to change some properties or selectors you can change them by modifying the scss files that you will find into de sass folder. In fact we have separated the selectors and values into the components folder so it should be easier to find wat you are searching. Normally you will want to modify the values into _variables.css file located into the settings folder.

To set gulp working to listen into the changes you will only need to keep running this command into a tab of the terminal into the booterfly folder:

$ gulp

Yeah, that simple! :)

Use it into your projects

In order to use Booterfly into your projects we provide a basic structure that you can use in your projects.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <title>Booterfly - Basic structure</title>

    <!-- Booterfly -->
    <link href="css/booterfly.css" rel="stylesheet">
  </head>
  <body>
    <h1>Booterfly is setup!</h1>

    <!-- jQuery (necessary for Booterfly's scripts) -->
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>
    <!-- Include Booterfly scripts -->
    <script src="js/booterfly.js"></script>
  </body>
</html>
